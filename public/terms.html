Termos de uso demonstrativos

Este projeto e uma simulacao para portfolio. Nenhuma operacao representa movimentacao financeira real.

Uso permitido

O sistema pode ser utilizado para avaliacao tecnica, demonstracao de interface e estudo de arquitetura.

Limitacoes

Dados, saldos, cartoes, investimentos e emprestimos sao demonstrativos. Para uso real, seriam necessarias integracoes reguladas, auditoria, seguranca adicional e infraestrutura propria.

Responsabilidade

Este software nao deve ser usado como banco real sem adequacoes legais, tecnicas e operacionais.
