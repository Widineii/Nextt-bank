Politica de privacidade demonstrativa

Esta pagina descreve como o projeto trataria dados em um ambiente real.

Dados coletados

Nome, e-mail, CPF, telefone, endereco e informacoes de operacoes sao usados apenas para demonstracao local.

Armazenamento

No modo demo, os dados ficam em JSON local dentro da pasta `data/`. Em producao, o projeto deve usar banco seguro, criptografia, backup e controle de acesso.

LGPD

Para uso real, seria necessario implementar consentimento, remocao de dados, portabilidade, base legal e registro de tratamento.
